# Phase 7: Passkeys (WebAuthn/FIDO2)

## Overview
Implement passkey (WebAuthn/FIDO2) support for passwordless authentication and device management.

## Requirements
- Register passkeys (WebAuthn/FIDO2) for users
- List, remove, and manage passkey devices
- Backend API for challenge, attestation, device CRUD
- Frontend UI for passkey management in settings
